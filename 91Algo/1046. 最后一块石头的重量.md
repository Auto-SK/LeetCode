## 方法一：大顶堆

### 算法流程

大顶堆

### 复杂度分析

* 时间复杂度：O(nlogn)
* 空间复杂度：O(n)

### 代码

``` python
class Solution:
    def lastStoneWeight(self, stones: List[int]) -> int:
        h = [-stone for stone in stones]
        heapq.heapify(h)

        while len(h) > 1:
            a, b = heapq.heappop(h), heapq.heappop(h)
            if a != b:
                heapq.heappush(h, a - b)
        return -h[0] if h else 0
```
