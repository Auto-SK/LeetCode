1. Sort

```python
class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        return sorted(s) == sorted(t)
```



2. Hash

```python
class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        if (len(s) != len(t)):
            return False
        count = 26 * [0]
        for i in range(len(s)):
            count[ord(s[i]) - ord('a')] += 1
        for i in range(len(t)):
            count[ord(t[i]) - ord('a')] -= 1
            if count[ord(t[i]) - ord('a')] < 0: # 小于说明t比s多，不能用大于或不能于，因为没遍历完
                return False
        return True
```
